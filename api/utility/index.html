Utility Functions

Sampling functions.

Function Name

Description

nphi()

Computes the number of \(\phi\).

ntheta()

Computes the number of \(\theta\) samples.

j_bandlimit()

Computes the band-limit of a specific wavelet scale.

L0_j()

Computes the minimum harmonic index supported by the given wavelet scale \(j\).

n_px()

Returns the number of spherical pixels for a given sampling scheme.

n_lm()

Returns the number of harmonic coefficients at bandlimit L.

n_lm_scal()

Computes the total number of harmonic coefficients for scaling kernels \(\Phi_{\el m}\)

n_lmn_wav()

Computes the total number of Wigner coefficients for directional wavelet kernels \(\Psi^j_{\el n}\).

n_gamma()

Computes the number of \(\gamma\) samples for a given sampling scheme

n_scal()

Computes the number of pixel-space samples for scaling kernels \(\Phi\).

n_wav()

Computes the total number of pixel-space samples for directional wavelet kernels \(\Psi\).

n_wav_j()

Number of directional wavelet pixel-space coefficients for a specific scale \(j\).

n_wav_scales()

Evalutes the total number of wavelet scales.

elm2ind()

Convert from spherical harmonic 2D indexing of \((\ell,m)\) to 1D index.

elmn2ind()

Convert from Wigner space 3D indexing of \((\ell,m, n)\) to 1D index.

Description
